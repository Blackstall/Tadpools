# Tadpools — System Logic

## Core Principle
The system uses structured swarm intelligence:
- Multiple agents analyze independently
- Agents share findings
- Agents challenge each other
- Consensus emerges
- Policy engine enforces final decision

## Workflow
1. Intake
2. Extraction
3. Swarm Activation
4. First Pass Analysis
5. Shared Memory
6. Challenge Phase
7. Consensus Formation
8. Policy Engine
9. Final Decision
10. Cleanup

## Agents
- Nature of Business Agent
- Registration Age Agent
- Document Authenticity Agent
- Existence Verification Agent
- Name Matching Agent
- Beneficiary Consistency Agent
- Historical Suspicion Agent
- Skeptic Agent
- Prosecutor Agent
- Chair Agent

## Decision Logic
- No high risk + low score → Approve
- Medium risk → Manual Review
- High risk → Escalate / Reject
